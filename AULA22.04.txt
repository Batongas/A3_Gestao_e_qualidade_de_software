Encapsulamento 

Identificador de acesso
 private - sem acesso pelas outras classes
 public - publico para as outras classes
 protect - publico apenas para a mesma hierarquia
 default - publico apenas para as classes do mesmo pacote/pasta


EXEMPLO
 
 public class Turma{
   
    private String nome;
    
    public Turma (String nome){ -> construtor
         this.nome = nome;
    }

    public String getNome(){ -> de acesso
        return nome;
    }

    public void setNome(String Nome){ -> modificador
        this.nome = nome;
    }
}
